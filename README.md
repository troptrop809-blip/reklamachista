# РекламоЧист Pro

Финальная версия сайта для проверки рекламных текстов на иностранные слова, англоязычные фразы и латиницу.

## Что внутри
- современный интерфейс
- API проверки текста
- загрузка TXT-файлов
- история отчетов
- статистика
- словарь из 700+ слов
- набор из 100+ популярных маркетинговых фраз

## Запуск
```bash
npm install
npm start
```

Открыть:
```bash
http://localhost:3000
```
